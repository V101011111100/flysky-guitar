# FlySky GitBook - Tài liệu vận hành

Tài liệu này dành cho 2 nhóm người dùng chính: Admin và Nhân viên vận hành.

Mục tiêu là hướng dẫn thao tác thực tế theo luồng công việc hằng ngày, bao gồm frontend, backoffice và các quy trình phối hợp liên phòng ban.

## Mục lục nhanh
- Bắt đầu tại: [Tổng quan phạm vi](./01-overview.md)
- Điều hướng đầy đủ: [SUMMARY](./SUMMARY.md)

## Phạm vi ưu tiên
- P0: Bảo mật, phân quyền, đơn hàng, sản phẩm và tồn kho
- P1: Reviews, consultations, marketing, discount, SEO, POS, activity log
- P2: KPI nâng cao, playbook sự cố, governance release, FAQ nâng cao
